# Privacy Policy

ReviewGhost does not collect, store, or transmit any personal information.

## Data Processing

- **URLs**: Processed server-side during analysis only. Not stored.
- **Review content**: Fetched, parsed, and analyzed in memory. Not persisted.
- **Results**: Returned to the client only. Not stored on the server.

## No User Accounts

ReviewGhost does not require user registration, login, or any form of authentication.

## No Cookies

ReviewGhost does not set tracking cookies or use analytics cookies.

## No Third-Party Sharing

ReviewGhost does not share any data with third parties because it does not collect any data.

## External Requests

When you submit a URL for analysis, ReviewGhost makes an HTTP request to that URL to fetch publicly available review data. The target site may log this request (IP address, user agent) as with any web request.

## Contact

For privacy questions, open an issue on the project repository.
